== Description ==
Post list block for new (Gutenberg) editor.

Selection of posts can be made by post type, taxonomy, author (exclude also available). The number of displayed posts and offset can also be controlled.

== Installation ==
1. Upload \"news-blocks\" directory to the \"/wp-content/plugins/\" directory. Or install it via \"Plugins\" menu in WordPress.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Add \"Post list\" block in an editor
